I'm a Computer Engineering student passionate about building **backend infrastructure** and understanding **how computer systems work beneath the abstractions**.

My interests span operating systems, memory management, networking, concurrency, storage engines, and distributed systems. Rather than relying on frameworks, I enjoy learning from first principles and applying those ideas by building production-inspired software in Modern C++.
